# MinModifier

# MinModifier ⇐ Modifier

A MinModifier causes die rolls under a minimum value to be treated as the minimum value.

Kind: global class
Extends: Modifier
See: MaxModifier for the opposite of this modifier
Since: 4.3.0

  • MinModifier ⇐ Modifier
    • new MinModifier(min)
    • .min ⇒ Number
    • .min
    • .name ⇒ string
    • .notation ⇒ string

    • .toJSON() ⇒ Object
